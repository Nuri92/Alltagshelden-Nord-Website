# Alltagshelden Nord Website

Dies ist die Website für **Alltagshelden Nord** – ein Angebot für Alltagshilfe, Betreuung und Unterstützung im eigenen Zuhause.

## Projektbeschreibung

Die Website stellt die Leistungen von Alltagshelden Nord vor. Dazu gehören Alltagshilfe, Gesellschaft und Aktivierung, Begleitung, digitale Unterstützung, Hilfe bei Papierkram sowie Beratung zur Abrechnung über die Pflegekasse.

Die Seite ist als statische Website umgesetzt und besteht aus HTML, CSS und JavaScript.

## Technologien

- HTML
- CSS
- JavaScript
- Pico CSS
- Font Awesome Icons
- Unsplash-Bilder
- Google Maps Embed mit Zwei-Klick-Lösung
